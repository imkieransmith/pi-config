# Pi config and extensions
This repo is my personal Pi coding agent setup. It is mainly a collection of custom  extensions, tools, skills, and configuration that I use day to day.

A number of the extensions started from work shared by other Pi users. Where that is the case, I have linked back to the original extension in the relevant source file comments. From there I have tweaked, combined, or rewritten pieces to better fit my own workflow, preferences, and local setup.

The code here is intended as a working personal config rather than a polished package, but it may be useful as a reference for building or adapting Pi extensions.
